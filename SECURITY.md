# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability within this project, please report it by creating an issue on our GitHub repository:

[https://github.com/sitepack-io/sitepack-cli/issues](https://github.com/sitepack-io/sitepack-cli/issues)

For severe issues, we prefer an email to [support@sitepack.nl](mailto:support@sitepack.nl).

We take all security vulnerabilities seriously and will work to address them as quickly as possible.
